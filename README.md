# Retail Sales Performance Dashboard (Power BI)

## Overview
This project presents an interactive Power BI dashboard analyzing sales performance across products, regions, and customer segments.

## Dashboards
1. Performance Overview
2. Product Profitability
3. Regional & Segment Analysis
4. Executive Summary

## Key Features
- KPI monitoring for Sales, Profit, and Margin
- Product profitability analysis
- Country-wise performance comparison
- Sales trend analysis
- Interactive slicers for filtering

## Tools Used
- Power BI
- Data Modeling
- DAX
- Data Visualization
